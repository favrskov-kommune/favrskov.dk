
-- INTRODUCTION --

Linkit_panel_pages exptends the Linkit module with the possibility to link to
pages from page_manager.

-- DEPENDENCIES --

Linkit_panel_pages has two obvious dependencies, Linkit and Page manager
* Linkit <http://drupal.org/project/linkit>
* Page manager (included in ctools) <http://drupal.org/project/ctools>
