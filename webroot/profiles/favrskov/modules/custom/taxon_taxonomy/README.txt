---------------------------------------------------------------------
About taxon_taxonomy
---------------------------------------------------------------------

taxon_taxonomy incorporates an interface between Drupal and Halibut Taxon.

Halibut Taxon is an Open Source system for automatic classification of texts.
It relies on one or more taxonomies for the analysis of the texts,
see more at www.taxon.dk.

Make sure to read the license information in the documentation/ directory.

----------------------------------------------------------------------
Setting up taxon_taxonomy
----------------------------------------------------------------------
In Configuration under Content Authoring in Taxon Taxonomy Field you 
need to enter some information about your setup. Direct link
(admin/config/content/taxon-taxonomy).
