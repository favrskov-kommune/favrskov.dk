INTRODUCTION
------------

Content reminder module allows to setup e-mail notifications on any content
to remind content editors to check the consistency and/or relevancy of that
particular item.

Enable content types from admin/config/workflow/content_reminder, make sure
cron is running.

Give appropriate roles permission to set reminders. 

Create a content or edit existing and set a reminder.

INSTALLATION
------------

Standard Drupal module installation.
